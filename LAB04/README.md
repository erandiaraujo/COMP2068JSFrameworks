Nirvana - Band - Website
A modern, responsive website dedicated to the legendary rock band Nirvana, built with Express.js and EJS templates.
Features

Pages

Home: Overview of Nirvana and their legacy
Kurt Cobain: Lead vocalist and guitarist
Krist Novoselic: Bassist and co-founder
Dave Grohl: Drummer

Installation

Navigate to the project folder:
 cd nirvana-site
Install dependencies:
  npm install
Start the server:
  npm start
Open your browser and visit:
  http://localhost:3000
